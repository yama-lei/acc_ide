# ACC IDE v1.2.0

## 更新日志

### 主要更新
- 添加了撤回和重做功能
- 添加了符号面板，便于安卓快速输入符号
- 使用`Github Action`真正实现了C++的IO功能

### 修复
- 修复了临时目录下文件关闭后无法正确删除的bug
- 修复了新建文件名称的bug
- 修复了启动界面卡白屏的bug

## 下载

- [ACC_IDE_v1.2.0.apk](ACC_IDE_v1.2.0.apk)

## 安装说明

1. 下载APK文件到您的Android设备
2. 打开APK文件安装应用（可能需要允许安装未知来源的应用）
3. 安装完成后，点击图标启动应用

## 注意事项

- 该版本需要Android 5.0 (API 21) 或更高版本
- 首次启动应用时，请授予存储权限以便应用正常工作
- 如遇任何问题，请在GitHub提交issue
